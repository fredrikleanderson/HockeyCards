# HockeyCards

Obs! I hockeykorten finns dubbletter. Inkludera alltid dubbletterna om inget annat anges.

Uppgift 1
Skriv om TaskOne() så att den returnerar det totala antalet mål för alla spelare.

Uppgift 2
Skriv om TaskTwo() så att den returnerar det lag som Markus Näslund spelar i.

Uppgift 3
Skriv om TaskThree() så att den returnerar antalet kort där spelaren spelar i Anaheim Mighty Ducks.

Uppgift 4
Skriv om TaskFour() så att den returnerar namnet på den back som gjort flest antal mål.

Uppgift 5
Skriv om TaskFive() så att den returnerar antalet kort i albumet om man räknas bort dubbletter.

Uppgift 6
Skriv om TaskSix() så att den ändrar namnen på spelarna i Washington Capitals så att de inte längre är skriva i… capitals.
